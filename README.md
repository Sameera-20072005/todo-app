## Project Overview
The Todo App is a simple Python Flask application that allows users to perform CRUD (Create, Read, Update, Delete) operations on todo items. The application uses SQLite as the database.

## Getting Started
To get started with the project, follow these steps:
* Clone the repository
* Install the required dependencies using `pip install -r requirements.txt`
* Run the application using `python app.py`

## Features
* Create new todo items
* Read all todo items
* Update existing todo items
* Delete todo items

## Testing
Testing is required for this project. To run tests, use the command `python -m unittest discover -s tests`

## CI/CD Pipeline
The project uses a CI/CD pipeline to automate testing and deployment. The pipeline is configured to run on every push to the repository.

## Contributing
The Todo App is a public project and contributions are welcome. To contribute, fork the repository, make changes, and submit a pull request. Ensure that all tests pass before submitting a pull request.
